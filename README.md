# MyCobot
Cobot controll and motion recording example

# Motion Gui
Record duration (s) - nustatyti 0 s (Judesių įrašymas be laiko apribojimo). 

Play speed - Roboto greitis

Drag teach (check box) - aktyvuoja judesių įrašymą rankiniu būdu vedant robotą (atlaisvina variklius)

Play Continuous (check box) - interpoliacija per laiką be papildomų sustojimų (vientisas judesys, o ne judėjimas nuo taško iki taško)

Record - pilnas judesio įrašymas (nėra pilnai išpildytas)

Record Main steps - tarpinių judesių įrašymas, kuriems vėliau taikoma interpoliacija) aktyvavus šį rėžimą judesiai įrašomi pasirenkant Mark step (keyframe) - tokiu būdu įrašomi pagrindiniai taškai, o tarpinės pozicijos apskaičiuojamos

Vacuum ON, Vacuum OFF - vakuuminio gripperio aktyvavymo įrašymas
